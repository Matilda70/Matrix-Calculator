Основные операции над матрицами

  • Сложение (A + B): складываются матрицы одинакового размера.
  • Вычитание (A − B): вычитаются матрицы одинакового размера
  • Умножение (A × B): возможно, если число столбцов A равно числу строк B.
  • Транспонирование Transpose(A): строки становятся столбцами.
  • Детерминант det(A): только для квадратных матриц. Для СЛАУ показывает единственность решения.
  • Обратная (A⁻¹): существует, если det(A) ≠ 0.
  • Степень (A^k): матрица умножается сама на себя k раз (например, A², A³).

Инструкция

  1. Выберите размеры A и B, нажмите «Применить».
  2. Заполните элементы матриц A и B.
  3. Выберите операцию; результат отобразится ниже.

Матрицы и операции

Матрица — прямоугольная таблица чисел. Обозначаем большими круглыми скобками.

Пример: Матрица A размера 2×2: 1234 . Элементы матрицы A обозначаются aij, где i - номер строки, в которой находится элемент, j - номер столбца.

Сложение и вычитание

Операция сложения и вычитания матриц вводится для матриц одинакового размера. Суммой двух матриц Aij и Bij называется матрица Cij,где cij = aij + bij.Записывают так: C = A + B. Аналогично определяется разность матриц.

Пример:

1234 + 5678 = 681012

Умножение

Возможно, если число столбцов A равно числу строк B. Произведением двух матриц А(n x m) и В(m x k) называется матрица С(n x k), где сik = ai1 * b1k + ai2 * b2k + ... + aik * bk1 + aik * bk2 + ... + aik * bkk. Записывается так: C = A * B. a11a12a21a22 × b11b12b21b22 = (a11*b11 + a12*b21) (a11*b12 + a12*b22)(a21*b11 + a22*b21) (a21*b12 + a22*b22)

Пример:

1234 × 5678 = 19224350

Транспонирование

Строки становятся столбцами.

1234 => 1324

Элементарные преобразования

  1. Перестановка местами двух параллельных строк/столбцов матрицы.
  2. Умножение всех элементов ряда матрицы на число, отличное от 0.
  3. Прибавление к одному ряду матрицы параллельного ряда, умноженного на число, отличное от 0.

Детерминант, обратная, ранг, степень

  • det(A) — числовой коэффициент который показывает, во сколько раз линейное преобразование изменяет объём (или площадь) пространства, а также указывает на сохранение или изменение его ориентации.

    Пример:

    det 1234 = (1 * 4) - (3 * 2) =−2.
  • A⁻¹ существует, если det(A)≠0. Произведение A и A⁻¹ равно единичной матрице.
  • rank(A) — число линейно независимых строк/столбцов матрицы. Rank 1224 = 1.
  • A^k — умножение матрицы самой на себя определенное количество раз (A², A³, …).
